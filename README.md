# AnonFund

Monorepo for the AnonFund platform.

## Workspace

- `apps/blockchain`: Smart contracts and deployment scripts.
- `apps/frontend`: Next.js frontend.
- `packages/circuits`: Circuits package.
- `packages/config`: Shared config package.

## Quick Start

1. Install dependencies from the repo root:

```bash
pnpm install
```

2. Run frontend app:

```bash
pnpm --filter @anonfund/frontend dev
```

3. Open:

`http://localhost:3000/connect`
